# Arka Plan Görselleri / Background Images

Bu klasör İmsakiye uygulaması için arka plan görsellerini içerir.

## Dosyalar

### Ana Görseller
- `mosque_blue.jpg` - Mavi Cami
- `mosque_night.jpg` - Gece Cami görüntüsü
- `kaaba.jpg` - Kabe
- `medina.jpg` - Medine manzarası
- `islamic_pattern.jpg` - İslami geometrik desen
- `ramadan_lantern.jpg` - Ramazan feneri
- `crescent_moon.jpg` - Hilal ay

### Küçük Resimler (thumbs/)
Her ana görsel için 200x200px küçük resim.

## Kullanım

Görseller mobil uygulamada arka plan olarak kullanılır.

## Telif Hakkı

Tüm görseller telif hakları serbest kaynaklardan (Unsplash, Pexels, Pixabay) alınmıştır.
