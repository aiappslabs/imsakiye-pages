# Ezan Sesleri / Adhan Sounds

Bu klasör İmsakiye uygulaması için ezan sesi dosyalarını içerir.

## Dosyalar

- `ezan_mecca.mp3` - Mekke ezan sesi
- `ezan_medina.mp3` - Medine ezan sesi
- `ezan_istanbul.mp3` - İstanbul ezan sesi
- `ezan_ankara.mp3` - Ankara ezan sesi

## Kullanım

Ses dosyaları mobil uygulama tarafından bildirimlerle birlikte kullanılır.

## Telif Hakkı

Tüm ses dosyaları telif hakları serbest kaynaklardan alınmıştır.
